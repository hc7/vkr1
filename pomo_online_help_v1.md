# Pomogator — Руководство пользователя веб-интерфейса

Веб-интерфейс доступен по адресу **http://\<ip-устройства\>:5000** (или через mDNS:
`http://pomo-gator-XXXXXX.local:5000`).  
Интерфейс состоит из четырёх страниц, переход между которыми осуществляется через
панель навигации в верхней части каждой страницы.

---

## Панель навигации

На каждой странице в правом верхнем углу отображается одна и та же панель:

| Кнопка | Действие |
|---|---|
| **POMO** (зелёная, иконка трансляции) | Мигает индикатором активности на устройстве в течение 5 секунд — подтверждает доступность устройства |
| **Upload** | Перейти к менеджеру файлов прошивок |
| **WiFi** | Перейти к настройкам Wi-Fi |
| **Self-update** | Перейти к странице обновления Pomogator |
| **Main** *(только на вложенных страницах)* | Вернуться на главную страницу |

---

## Главная страница — Загрузка прошивки (`/`)

Используется для прошивки ADCU / TBOX по протоколу DoIP.

### Параметры загрузки

| Поле | Описание |
|---|---|
| **Target IP** | IP-адрес целевого блока управления (по умолчанию `192.168.9.4`). Можно изменить прямо в поле. |
| **Firmware** | Текущий выбранный файл прошивки. Поле только для чтения — нажмите **Change** или перейдите в **Upload**, чтобы выбрать другой файл. |
| **ISO version 3** | Установите флажок для блоков управления, использующих ISO 14229-3 (UDS on DoIP v3). Для стандартного v2 флажок не нужен. |

### Запуск загрузки

1. Укажите правильный **Target IP**.
2. Убедитесь, что отображается нужный файл **Firmware**.
3. При необходимости установите флажок **ISO version 3**.
4. Нажмите **DEPLOY** (зелёная кнопка).

Во время выполнения кнопка меняется на **CANCEL** (красная).  
Нажмите **CANCEL** для прерывания в любой момент.

### Панель прогресса

- **Phase** — текущий этап (например, *Upload*, *Activate*, *Reset*).
- **Прогресс-бар** — процент выполнения; анимирован во время работы, зелёный при успехе, красный при ошибке.
- **Last message** — последнее сообщение от машины состояний загрузки.
- Ссылка **Deploy log** для скачивания лога появляется после завершения (успешного или с ошибкой).

### Лог

Прокручиваемый лог в нижней части страницы отображает все сообщения в реальном времени.  
Нажмите **Clear** для очистки отображения (серверный лог при этом не затрагивается).

### Карточка «ADCU Versions»

Отображает версии прошивок, запущенных на подключённом блоке управления (MCU, SOC, J3).

| Индикатор | Значение |
|---|---|
| Значок **CAN OK** (зелёный) | Keep-alive-фреймы успешно достигают блока управления по `can1` |
| Значок **No comm** (красный) | CAN-связь не обнаружена |

Версии считываются автоматически каждые 30 секунд при активной связи.
При потере связи последние известные значения остаются видимыми и выделяются красным цветом.

## Страница Upload — Файлы прошивок (`/upload`)

Управление файлами прошивок, хранящимися на устройстве в `/opt/firmware/`.

### Загрузка файла

1. Нажмите **Choose File** и выберите файл `.zip`, `.bin` или `.hex`.
2. Нажмите **Upload**.

Браузер вычисляет контрольную сумму MD5 перед отправкой и проверяет её на сервере.
Прогресс-бар показывает процент загрузки. По завершении файл появится в списке.

### Список файлов

Каждая строка содержит имя файла, размер и дату изменения.

| Элемент управления | Действие |
|---|---|
| Переключатель (радиокнопка) | Выбрать этот файл для загрузки (отображается на главной странице) |
| Иконка **корзины** | Удалить файл (запрашивает подтверждение) |

Одновременно можно выбрать только один файл. Выбор применяется немедленно.

В нижней части страницы расположена небольшая строка статуса загрузки — она отражает
текущий прогресс активной операции прошивки, чтобы не уходить со страницы.

## Страница WiFi — Настройки Wi-Fi (`/wifi`)

Настройка подключения **WLAN-CLIENT** — соединения, по которому устройство подключается
к существующей Wi-Fi-сети (точка доступа на производстве, офисный роутер и т. д.).

### Текущий статус подключения

Карточка в верхней части отображает:

- **SSID** — название сети, для которой настроен профиль подключения.
- **Mode** — `infrastructure` (клиент) или `ap` (точка доступа, для встроенного AP).
- Значок **Connected / Disconnected** (подключено / отключено).
- Предупреждение **Unsaved changes** — появляется, если профиль был изменён в памяти,
  но ещё не сохранён на диск. Нажмите **Save to disk** в этом предупреждении, чтобы
  сохранить настройки.

### Обновление подключения

1. Введите **SSID** целевой сети или нажмите кнопку **поиска** (лупа) рядом с полем
   SSID, чтобы увидеть доступные сети.
2. Введите **Password** (пароль).
3. Выберите опции:
   - **Save to disk** — записать профиль на SD-карту (сохраняется после перезагрузки).
   - **Restart connection** — немедленно переподключиться.
4. Нажмите **Update**.

### Доступные сети

Нажмите иконку лупы (или дождитесь автоматического сканирования раз в 30 секунд),
чтобы увидеть список ближайших Wi-Fi-сетей с уровнем сигнала, типом защиты и
диапазоном частот.  
Нажмите **Use** в любой строке, чтобы скопировать SSID в форму обновления.

## Страница Self-update — Обновление Pomogator (`/self-update`)

Замена программного обеспечения Pomogator на устройстве из `.zip`-пакета
(формируется командой `./pack.sh` из репозитория).

> **Внимание:** применение обновления останавливает и перезапускает веб-сервис.
> Браузер кратковременно потеряет соединение и перезагрузится автоматически.

### Шаг 1 — Загрузка пакета

1. Нажмите **Choose File** и выберите файл `pomogator-<V>.<B>.zip`.
2. Нажмите **Upload**.

Прогресс-бар отслеживает загрузку. При успехе появляется карточка **Staged package**.

### Шаг 2 — Подготовленный пакет

Проверьте данные перед применением:

| Поле | Описание |
|---|---|
| **File** | Имя файла пакета |
| **Size** | Размер файла |
| **MD5** | Контрольная сумма MD5 загруженного файла |
| **Version** | Номер версии Pomogator из пакета |
| **Build** | Монотонный номер сборки из пакета |

### Шаг 3 — Применение

Нажмите **Apply update (restarts service)** и подтвердите диалог.

Карточка **Apply status** отображает ход выполнения. После завершения обновления
страница перезагружается автоматически. Если сервис перезапускался дольше обычного,
обновите страницу вручную.

## Индикаторы (аппаратная справка)

| Индикатор | GPIO | Цвет | Поведение |
|---|---|---|---|
| Comm | 1 (BCM) | Зелёный | Горит, пока активна CAN-связь с блоком управления |
| Activity | 12 (BCM) | Красный | Мигает 5 секунд при нажатии кнопки **POMO** |
